# react-app
This is a react app without the boiler plate. Best for basic projects

For windows

```
git clone https://github.com/SarthakNarayan/react-app.git
cd react-app && npm install
rmdir /s /q .git
```

For linux

```
git clone https://github.com/SarthakNarayan/react-app.git
cd react-app && rm -rf .git && npm install
```
